Zum Starten von Backend, User-Frontend und Admin-Frontend
geben Sie bitte folgenden Befehl in die Kommandozeile ein:

docker compose up --build

Wenn die Information "BirdParkApplication has been started" in der Konsole erscheint,
ist die Software erfolgreich gestartet.

Die Anwendung ist unter folgenden Adressen erreichbar:
- User-Frontend: http://localhost:8081
- Admin-Frontend: http://localhost:8082
- Swagger-UI: http://localhost:8080/swagger-ui/index.html#/
